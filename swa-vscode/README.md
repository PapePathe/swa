# Swa VS Code Extension

This extension provides syntax highlighting and basic language support for the Swa programming language.

## Installation

1.  Copy the `swa-vscode` directory to your VS Code extensions folder:
    -   **Windows**: `%USERPROFILE%\.vscode\extensions`
    -   **macOS/Linux**: `~/.vscode/extensions`
2.  Restart VS Code.

## Features

-   Syntax highlighting for Swa keywords, types, strings, and numbers.
-   Comment support (`//`).
-   Bracket matching and auto-closing.
